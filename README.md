# Ecos del Tiempo

Proyecto final Django estilo blog.

## Funcionalidades

- Registro y login de usuarios
- Perfil de usuario
- Creación de ecos (posts)
- Edición y eliminación de ecos
- Mensajería entre usuarios
- Página About
- CKEditor para contenido

## Instalación

1. Clonar repositorio

git clone https://github.com/usuario/ecos-del-tiempo

2. Instalar dependencias

pip install -r requirements.txt

3. Migrar base de datos

python manage.py migrate

4. Ejecutar servidor

python manage.py runserver
